## 🚧 Current Projects

### 🎬 Stanza *(Private WIP)*
> A modern self-hosted media platform focused on flexibility and control

Built because I wanted more customization than existing solutions provided, and to challenge myself to build a full platform.

- Playback resolver system (stream selection, audio/subtitles, HDR)
- Session-based playback flow (preparing → ready → active)
- Watch tracking & resume support
- Next.js + backend API architecture
- Planned plugin system + native apps

---

### 🧱 OHM Engine *(Private WIP)*
> A custom Java game engine focused on performance and flexibility

Started as wanting to make a game, then turned into building the engine itself. Chose Java to explore how far I could push it.

- OpenGL rendering pipeline (LWJGL)
- Windowing + input system (GLFW)
- Modular multi-module architecture
- Custom engine systems (rendering, core structure)
- Planned Vulkan support
